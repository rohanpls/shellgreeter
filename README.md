## Installation

```bash
# Clone this repo anywhere you want!
$ git clone https://github.com/rohanpls/shellgreeter.git

# If you use Bash
$ echo "python3 ~/shellgreeter/greeter.py" >> ~/.bashrc

# If you use Zsh
$ echo "python3 ~/shellgreeter/greeter.py" >> ~/.zshrc

```
